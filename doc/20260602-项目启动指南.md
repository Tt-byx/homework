## 六、可能遇到的问题

### 1. Java 版本兼容性问题
pom.xml 中配置的 `<java.version>17</java.version>`，但你本机是 Java 25。Spring Boot 3.3.5 理论上兼容 Java 17-21，Java 25 可能会有警告但不影响运行。如果遇到编译错误，可以尝试：

```powershell
# 在 pom.xml 中修改 java.version 为 25
# 或者使用 --add-opens 参数启动
```

### 2. MySQL 连接失败
确认 `application.yml` 中的数据库密码是否正确（当前为 `123456`），确认 MySQL 服务正在运行。

### 3. Python 后端报错
确认 `.env` 文件中的 API Key 是否有效，确认虚拟环境已激活，依赖已安装（`uv sync` 或 `pip install -r requirements.txt`）。

### 4. 端口被占用
如果某个端口被占用，可以用以下命令查看并释放：

```powershell
# 查看端口占用
netstat -ano | findstr :8080

# 杀死进程（替换 PID）
taskkill /PID <PID> /F
```
