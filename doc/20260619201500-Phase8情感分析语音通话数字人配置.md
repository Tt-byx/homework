|------|------|
| `services/sentiment_service.py` | MiMo API 情感分析，输出 sentiment/emotion/expression |

| 文件 | 说明 |

**情感映射表：**
| emotion | Live2D 表情 |
|---------|------------|
| happy/excited/grateful | Smile |
| surprised | Star |
| confused | Circle |
| angry | Angry |
| worried/sad | Cry |

### 8.2 半双工语音通话

**前端新建：**
| 文件 | 说明 |
|------|------|
| `views/visitor/VoiceCallView.vue` | 全屏通话页面 |

**交互流程：**
1. 点击"通话模式" → 进入全屏 Live2D 页面
2. 点击"按住说话" → MediaRecorder 录音
3. 松手 → WebSocket 发送音频 → AI 流式回复
4. AI 播放完毕 → 自动恢复监听（循环对话）
5. 点击"结束通话" → 回到聊天页面

**前端修改：**
| 文件 | 说明 |
|------|------|
| `components/chat/ChatInput.vue` | 新增"通话模式"按钮 |
| `router/index.js` | 新增 `/call` 路由 |
